******* SISTEMA DE PERMISOS Y VACACIONES CHEDRAUI  MERCADERIAS ****

desarrollado en vb.net  mysql 5.0
DESCRIPCION:

Permite la administracion de permisos y vacaciones para los empleados del corporativo xalapa en el departamento de mercaderias.

* control de empleados y datos personales()
* gestion de puestos (gerentes- subgenrentes- jefes- empleados).
* control de dias por año.
* generacion de pdf con los datos para RH especificando dias de vacaciones.
* generacion de permisos de salida por hora y dias.
* bitacora de permisos de salida, vacaciones y incapacidad.
* reporte en formato hoja de calculo Excel y PDF.
* base de datos centralizada (Cliente - Servidor).
